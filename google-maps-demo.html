🔧 Google Maps API セットアップ手順

  1. Google Cloud Console にアクセス: https://console.cloud.google.com/
  2. 新しいプロジェクトを作成するか、既存のプロジェクトを選択
  3. 「APIs & Services」→「Library」で「Maps JavaScript API」を有効化
  4. 「APIs & Services」→「Credentials」でAPIキーを作成
  5. 下記のHTML内の「YOUR_API_KEY」を実際のAPIキーに置き換え
  6. 必要に応じてAPIキーの制限を設定（HTTPリファラー、IPアドレスなど）

注意: 本番環境では必ずAPIキーを適切に保護してください。

⚙️ 実装コード例

以下のコードを使用してGoogle Mapsを統合できます：

<!-- Google Maps JavaScript API --> <script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap&libraries=geometry"> </script> <script> function initMap() { // 地図の初期化 const map = new google.maps.Map(document.getElementById('map'), { zoom: 12, center: { lat: 35.6580, lng: 139.7016 }, // 品川駅 mapTypeId: 'roadmap' }); // ルート表示 const directionsService = new google.maps.DirectionsService(); const directionsRenderer = new google.maps.DirectionsRenderer(); directionsRenderer.setMap(map); // ルート計算 directionsService.route({ origin: { lat: 35.6894, lng: 139.6917 }, // 新宿駅 destination: { lat: 35.4647, lng: 139.6224 }, // 横浜駅 travelMode: google.maps.TravelMode.DRIVING }, (response, status) => { if (status === 'OK') { directionsRenderer.setDirections(response); } }); } </script>
